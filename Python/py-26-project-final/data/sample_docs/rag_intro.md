# RAG 技术简介

## 什么是 RAG？

RAG（Retrieval-Augmented Generation，检索增强生成）是一种结合信息检索和文本生成的 AI 技术。它通过在生成回答前先检索相关信息，来提高大语言模型（LLM）回答的准确性和可靠性。

## 核心优势

### 1. 减少幻觉

传统的 LLM 可能会"编造"不存在的信息。RAG 通过提供真实的参考资料，显著减少了这种"幻觉"现象。

### 2. 知识可更新

无需重新训练模型，只需更新知识库即可让系统获得最新信息。

### 3. 来源可追溯

每个回答都可以追溯到具体的文档来源，增强了答案的可信度。

## 工作原理

RAG 系统的工作流程通常包括：

1. **文档处理**: 将文档分割成小块（chunks）
2. **向量嵌入**: 将文本块转换为向量表示
3. **索引存储**: 将向量存储到向量数据库中
4. **查询处理**: 将用户问题转换为向量
5. **检索匹配**: 找到最相关的文档块
6. **生成回答**: 将检索到的内容作为上下文，让 LLM 生成回答

## 应用场景

- 企业知识库问答
- 客服机器人
- 技术文档助手
- 研究文献检索
- 法律法规查询
