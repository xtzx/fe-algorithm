# Sample Project

这是一个用于测试代码统计工具的示例项目。

## 项目结构

```
sample_project/
├── src/
│   ├── main.py      # 主模块
│   ├── utils.py     # 工具函数
│   └── api.js       # JavaScript 示例
├── tests/
│   └── test_main.py # 测试
└── docs/
    └── README.md    # 文档
```

## 包含的语言

- Python (.py)
- JavaScript (.js)
- Markdown (.md)

## 用途

使用 code-counter 工具统计此项目：

```bash
code-counter scan examples/sample_project
```
