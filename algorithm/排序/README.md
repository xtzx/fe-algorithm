## 📖 核心理念

1. **图解直觉优先**：先建立直觉，再看形式化证明
2. **场景驱动选型**：从需求出发，而非从算法出发
3. **工程化实现**：类型安全、可测试、可复用
4. **前端视角**：表格排序、搜索结果、大数据分片
