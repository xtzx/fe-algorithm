## 🗺️ 机器学习全景图

### 机器学习 vs 传统编程

```
传统编程：
  输入数据 + 规则 → 程序 → 输出结果
  （人工编写规则）

机器学习：
  输入数据 + 期望输出 → 算法 → 学习出规则（模型）
  （让机器从数据中学习规则）
```

### 机器学习任务分类

```
机器学习
├── 监督学习（有标签）
│   ├── 分类（Classification）
│   │   ├── 二分类：垃圾邮件检测、疾病诊断
│   │   └── 多分类：图像识别、情感分析
│   └── 回归（Regression）
│       └── 预测连续值：房价预测、销量预测
│
├── 无监督学习（无标签）
│   ├── 聚类（Clustering）
│   │   └── 用户分群、异常检测
│   └── 降维（Dimensionality Reduction）
│       └── 特征压缩、数据可视化
│
└── 强化学习（奖励信号）
    └── 游戏AI、机器人控制
```

### 本阶段覆盖的算法

| 类型 | 算法 | 应用场景 |
|------|------|---------|
| **线性模型** | Linear Regression | 回归基线 |
| | Logistic Regression | 二分类基线 |
| | Ridge / Lasso | 带正则化的回归 |
| **树模型** | Decision Tree | 可解释性要求高 |
| | Random Forest | 通用性强 |
| | XGBoost / LightGBM | 竞赛首选 |
| **无监督** | K-Means | 聚类 |
| | PCA | 降维 |
| | t-SNE / UMAP | 可视化 |

## 🧭 模型选择决策树

```
┌─ 有标签数据？
│  ├─ 是 → 监督学习
│  │  ├─ 目标是类别？→ 分类
│  │  │  ├─ 需要概率输出？→ 逻辑回归
│  │  │  ├─ 需要可解释性？→ 决策树
│  │  │  ├─ 追求最高精度？→ XGBoost/LightGBM
│  │  │  └─ 通用场景 → 随机森林
│  │  │
│  │  └─ 目标是数值？→ 回归
│  │     ├─ 简单线性关系？→ 线性回归
│  │     └─ 复杂非线性？→ 树模型
│  │
│  └─ 否 → 无监督学习
│     ├─ 分组？→ 聚类（K-Means/DBSCAN）
│     └─ 降维/可视化？→ PCA/t-SNE/UMAP
```

### 验证安装

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
import sklearn

print(f"NumPy: {np.__version__}")
print(f"Pandas: {pd.__version__}")
print(f"Scikit-learn: {sklearn.__version__}")

# 测试 sklearn
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier

iris = load_iris()
X_train, X_test, y_train, y_test = train_test_split(
    iris.data, iris.target, test_size=0.2, random_state=42
)
model = RandomForestClassifier(n_estimators=10, random_state=42)
model.fit(X_train, y_train)
print(f"测试准确率: {model.score(X_test, y_test):.2%}")
```
