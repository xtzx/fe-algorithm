# 🧭 方向选择

> 2024-2025 年 AI 领域热门方向分析与定位

---

## 四大热门方向

### 1. LLM 应用工程师（RAG / Agent）

```
定位：用大模型构建实际应用的工程师

核心技能：
- Prompt Engineering（必须精通）
- RAG 系统设计与优化
- Agent 框架（LangChain/LlamaIndex/LangGraph）
- API 设计与服务化
- 向量数据库使用

典型工作内容：
- 设计和实现企业知识库、智能客服
- 构建 AI Agent 自动化工作流
- RAG 系统调优（召回率、准确率）
- 与业务团队对接需求

薪资范围（2024 年国内）：
- 初级（1-2年）：25-40K/月
- 中级（3-5年）：40-60K/月
- 高级（5年+）：60-100K/月

发展空间：
- 横向：产品经理、解决方案架构师
- 纵向：AI 技术负责人、CTO
- 创业：AI 应用 SaaS

适合人群：
✅ 有后端/全栈开发经验
✅ 对业务场景敏感
✅ 喜欢快速迭代和落地
```

### 2. AI Infra 工程师（部署 / 推理优化）

```
定位：保障 AI 系统高效稳定运行的基础设施工程师

核心技能：
- 推理引擎（vLLM/TensorRT/Triton）
- 模型量化与优化
- Kubernetes + GPU 调度
- 性能调优（吞吐/延迟）
- 监控与运维

典型工作内容：
- 搭建高可用推理服务
- 优化推理性能（吞吐量提升 X 倍）
- GPU 资源调度与成本优化
- CI/CD 流水线建设

薪资范围（2024 年国内）：
- 初级（1-2年）：25-35K/月
- 中级（3-5年）：35-55K/月
- 高级（5年+）：55-90K/月

发展空间：
- 横向：MLOps、DevOps
- 纵向：基础设施架构师
- 云厂商：AI 平台产品

适合人群：
✅ 有运维/后端经验
✅ 对系统性能敏感
✅ 喜欢底层优化
```

### 3. 算法工程师（微调 / 预训练）

```
定位：提升模型能力的算法研究与实践者

核心技能：
- 深度学习框架（PyTorch 必须）
- Transformer 架构深入理解
- 微调技术（LoRA/QLoRA/全量）
- 数据工程（清洗/标注/增强）
- 评估体系设计

典型工作内容：
- 针对特定任务微调模型
- 数据集构建与质量把控
- 模型评估与迭代
- 跟踪最新研究并复现

薪资范围（2024 年国内）：
- 初级（1-2年）：30-45K/月
- 中级（3-5年）：45-70K/月
- 高级（5年+）：70-120K/月

发展空间：
- 横向：研究科学家
- 纵向：算法负责人
- 学术：继续深造

适合人群：
✅ 数学/统计背景好
✅ 喜欢研究和创新
✅ 能接受较长的反馈周期
```

### 4. 行业 AI 解决方案

```
定位：将 AI 能力落地到具体行业的专家

核心技能：
- 行业知识（医疗/法律/金融/教育等）
- LLM 应用开发
- 需求分析与方案设计
- 项目管理

典型工作内容：
- 理解行业痛点，设计 AI 解决方案
- 与客户沟通需求
- 定制化开发与交付
- 售前支持与演示

薪资范围（2024 年国内）：
- 初级（1-2年）：20-35K/月
- 中级（3-5年）：35-55K/月
- 高级（5年+）：55-100K/月+（可能有提成）

发展空间：
- 横向：行业顾问
- 纵向：解决方案总监
- 创业：垂直 SaaS

适合人群：
✅ 有行业背景或兴趣
✅ 沟通能力强
✅ 喜欢解决实际问题
```

## 方向选择决策树

```
你更喜欢...

偏工程实现？
├── 是 → 关注性能和稳定性？
│        ├── 是 → AI Infra 工程师
│        └── 否 → LLM 应用工程师
└── 否 → 关注模型本身？
         ├── 是 → 算法工程师
         └── 否 → 行业 AI 解决方案
```

## 推荐组合策略

### 策略 A：主攻 LLM 应用 + 辅修 AI Infra

```
适合：想快速入行，追求稳定就业

主线技能：
- RAG / Agent 开发
- Prompt Engineering
- FastAPI 服务化

辅助技能：
- Docker 部署
- 基础推理优化
- 监控运维

作品集建议：
1. 企业知识库 RAG 系统（旗舰）
2. 多 Agent 自动化工具（旗舰）
3. 高性能推理服务部署（亮点）
```

### 策略 B：主攻算法 + 辅修 LLM 应用

```
适合：想深入技术，追求高薪资

主线技能：
- 模型微调
- 数据工程
- 评估体系

辅助技能：
- RAG 基础
- 简单应用开发

作品集建议：
1. 领域模型微调 + 评估报告（旗舰）
2. 数据集构建工具（亮点）
3. 简单 RAG 应用（补充）
```

### 策略 C：行业 + LLM 应用

```
适合：有行业背景，想转型 AI

主线技能：
- 行业知识深耕
- LLM 应用开发
- 需求分析

辅助技能：
- 基础 RAG
- 简单部署

作品集建议：
1. 行业垂直 AI 应用（旗舰）
2. 行业知识库（旗舰）
3. 行业数据分析工具（补充）
```
